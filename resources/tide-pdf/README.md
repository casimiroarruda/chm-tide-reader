# Tábuas de Maré

## Usage

1. First create a new folder here with the year, for example: `2026`.

2. Go to [https://www.marinha.mil.br/chm/dados-do-segnav/dados-de-mare-mapa](https://www.marinha.mil.br/chm/dados-do-segnav/dados-de-mare-mapa) and select the desired year. 

3. Finally, select the desired location and download the tide tables (PDF Icon) for the desired year and insert them in the folder created in the first step.
